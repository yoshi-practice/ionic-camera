# ionic-camera
This is an ionic application that is able to take a picture for android and iOS .

---

## How to setup ?

```:bash
$ ionic cordova platform add android ios
$ ionic cordova plugin add cordova-plugin-camera cordova-base64-to-gallery
$ npm install
```

